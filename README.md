# YOLOKAN
YOLOKAN-11 est une architecture hybride innovante conçue pour le diagnostic automatisé du paludisme sur frottis sanguins. Ce projet intègre les Réseaux de Kolmogorov-Arnold (KAN) a base des polynôme de Tchebychev au sein de l'architecture YOLO11 pour surpasser les limites des convolutions classiques.
